这个游戏是一个经典的扫雷游戏，运行./MineSweeper/MineSweeper.exe程序可以打开游戏

游戏的主场景标题“经典扫雷”是一个从上掉下来的动画，初次运行时可能会卡顿
主场景有三个文字按钮，按钮没有任何边界，按钮全是文字，点击文字，按钮跳动后就会进入对应的游戏界面

游戏界面的开始菜单里仅有一个退出菜单项，单击可实现退出程序
另外，游戏界面顶部左端有一个计时器(记录游戏时间，在游戏胜利后可以显示出完成游戏的用时)
顶部右端有一个统计剩余地雷的标签，可以实时统计还有多少地雷剩余
顶部中间是一个文字按钮，点击实现返回(但不会跳动)

最后，游戏没有设计任何音效，所以，如果没听到声音，那不是你声卡的问题哦...

以上是基本介绍，看完之后就可以尽情的去扫雷了，come...

左键点击扫雷，右键点击插旗，右键双击变问号，旗子和问号不允许直接点开
需要右键直到恢复原始才可以点开

另外，要想运行源代码的话，一定要注意中文路径哦...
